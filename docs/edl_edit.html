EDL Edit Window

You can bring up the EDL Edit window by doing any of:
  • Opening up the window from the main toolbar's
    W
    button.
  • By using
    RMB->Window->EDL Edit.

The EDL Edit Window allows you to see a reel as a timeline EDL and allows you to perform non-destructive edits on the movie files loaded.


An EDL is basically a list of images stored under a particular name.  This list of images can be reordered simply by doing drag and drop of the image in the Reel window or in the EDL Edit window.  You can also quickly switch from one image or movie to another by clicking on any item on the EDL.

mrViewer allows you to zoom into a timeline bar of the EDL by pressing the 'f' key. You can also zoom out to see all bars of the EDL by pressing 'a'.

The Editing of the EDL is done by selecting with the RMB the start or ending of an EDL timebar and dragging left or right according to where you want more or less frames to show. While dragging the current frame is displayed in the main viewer window.

Currently you cannot reliably do edits across the network to a remote mrViewer. This is a limitation that shall be removed in the future.
