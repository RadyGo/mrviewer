The FFMPEG Utility


mrViewer comes bundled with a copy of ffmpeg, which is a command-line utility used to create movies among other things. The current version supports OpenEXR non-tiled images but only in zip, rle and non compressed modes.
You can create mp4 movie files from images with the following syntax:
ffmpeg -layer Specular -i image.%04d.exr test.mp4

If not working with a multi-layered openexr file, the command can be simplified to:
ffmpeg -i image.%04d.exr test.mp4
